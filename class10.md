# Class 10 Exercise

## Easy
#### Question: LeetCode - Valid Parentheses
[link](https://leetcode.com/problems/valid-parentheses/) 

#### Example:
```
Input: s = "()"
Output: true
```
```
Input: s = "()[]{}"
Output: true
```
```
Input: s = "(]"
Output: false
```
```
Input: s = "([)]"
Output: false
```
```
Input: s = "{[]}"
Output: true
```

## Hard
#### Question: LeetCode - Regular Expression Matching
[link](https://leetcode.com/problems/regular-expression-matching/)

#### Example:
```
Input: s = "aa", p = "a"
Output: false
```
```
Input: s = "aa", p = "a*"
Output: true
```
```
Input: s = "ab", p = ".*"
Output: true
```
```
Input: s = "aab", p = "c*a*b"
Output: true
```
```
Input: s = "mississippi", p = "mis*is*p*."
Output: false
```
